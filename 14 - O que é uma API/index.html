O que é um API

  • Aplication Programming interface
  • Interface de programa de Aplicações

conjunto de programação que permte a contrução de sistemas que abstraem muita coisa de código.

EndPoints

nada mais são que urls, endereçõs da web que a gente vai chamar por exemplo vuscar uma lista de clientes, ou pra salvaum novo cliente em um banco de dados

4 tipos

  • Get
  • Post
  • Put
  • Delete
